# Wind turbine impact on cranes

**This webmap was created as MAAN7771 course work by Eliina Alroth, Laura Koivukoski and Lauri Nieminen.** 

The map depicts the impact of wind turbines on cranes' migration paths. The solution responded to challenge provided by Centre for Economic Development, Transport and the Environment in Southwest Finland
(Varsinais-Suomen ELY-keskus). 

Eplore the map: [https://utu-geospatial-labs.github.io/2024-webmap-windpower-and-cranes/index.html](https://utu-geospatial-labs.github.io/2024-webmap-windpower-and-cranes/index.html )
